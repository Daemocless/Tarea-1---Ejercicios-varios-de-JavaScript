# Tarea-1---Ejercicios-varios-de-JavaScript

Alumno: Kilian Rodriguez Duque

---

## Descripción
Este repositorio contiene 10 ejercicios básicos de JavaScript realizados en archivos HTML independientes. Cada ejercicio demuestra el uso de eventos, manipulación del DOM y lógica básica en JavaScript.

---

## Ejercicios

### Ejercicio 1: Mostrar un mensaje al hacer clic
Archivo: `ejercicio1.html`  
Descripción: Al pulsar un botón se muestra el mensaje "¡Hola Clase!".

---

### Ejercicio 2: Cambiar el color del texto
Archivo: `ejercicio2.html`  
Descripción: Cambia el color de un párrafo a rojo al hacer clic en un botón.

---

### Ejercicio 3: Contador de clics
Archivo: `ejercicio3.html`  
Descripción: Cuenta cuántas veces se pulsa un botón y muestra el resultado.

---

### Ejercicio 4: Mostrar la hora actual
Archivo: `ejercicio4.html`  
Descripción: Muestra la hora actual del sistema al pulsar un botón.

---

### Ejercicio 5: Saludo personalizado con input
Archivo: `ejercicio5.html`  
Descripción: Muestra un saludo personalizado con el nombre introducido por el usuario.

---

### Ejercicio 6: Calculadora de suma simple
Archivo: `ejercicio6.html`  
Descripción: Suma dos números introducidos en inputs y muestra el resultado.

---

### Ejercicio 7: Botón de ocultar / mostrar
Archivo: `ejercicio7.html`  
Descripción: Permite ocultar y mostrar un párrafo con un botón.

---

### Ejercicio 8: Cambiar imagen al hacer clic
Archivo: `ejercicio8.html`  
Descripción: Cambia la imagen mostrada al hacer clic en un botón.

---

### Ejercicio 9: Lista de tareas sencilla
Archivo: `ejercicio9.html`  
Descripción: Permite añadir elementos a una lista desde un input.

---

### Ejercicio 10: Semáforo interactivo
Archivo: `ejercicio10.html`  
Descripción: Simula un semáforo que cambia entre rojo, naranja y verde con un botón.
